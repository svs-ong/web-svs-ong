# Lab Work: Web Internship Homework

## Objective:

Create a basic **Home Page** with a **Navigation Bar** , **Hero Section** , **Articles Section** , and **Footer** . Also, set up placeholder pages for **Blog Post** , **Article** , and **Contact** . Focus on layout, components, and navigation. Styling is not the priority at this point.

## Steps:

- **Create the React App:**

  - Fork the repo from [this link](https://github.com/svs-ong/web-intership-homework) and clone it to your computer.

  - Create a new branch with your team name.

  - Create a new React app named `react-homework`.

- **Set Up Pages and Navigation:**

  - Set up the following pages:

    - **Home Page** (default page)

    - **Blog Post Page** (with placeholder content)

    - **Article Page** (with placeholder content)

    - **Contact Page** (with placeholder content)

  - Implement **React Router** for navigation between these pages.

- **Create Components:**

  - Split your app into main components:

    - **Navbar** : A simple navigation bar with links (e.g., Home, Articles, Blog Post, Contact).

    - **Hero Section** : A section with a heading and a description.

    - **Articles Section** : Display placeholder text (e.g., "Article 1", "Article 2").

    - **Article** : A component to represent a single article.

    - **Footer** : A simple footer.

    - **Additional Components** : Create others if needed.
