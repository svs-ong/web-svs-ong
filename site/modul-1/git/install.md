# Git Installation

Git is a program that runs on your computers. It needs to be installed before it can be used. To install Git, follow the steps below:

## Windows

1. Download Git from [git-scm.com](https://git-scm.com/).
2. Open the downloaded file and follow the installation steps.
3. Open CMD or Powershell and run the command `git --version` to verify if Git has been installed correctly.

Video: [Windows - Git Installation](https://www.youtube.com/watch?v=iYkLrXobBbA)


## Mac

1. Install [Homebrew](https://brew.sh/).
2. Open the terminal and run the command `brew install git`.
3. Run the command `git --version` to verify if Git has been installed correctly.

Video: [Mac - Git Installation](https://www.youtube.com/watch?v=26hzQ6KWhDM)

## Linux

1. Congratulations! You are a #based Linux user.
2. If you are on Linux, it means you already know how to install Git.
3. If not, open the terminal and run the command `sudo apt-get install git` for Ubuntu or the equivalent for your distribution.

> Many Linux distributions come with Git pre-installed, so you can run the command `git --version` to check if Git is already installed.
