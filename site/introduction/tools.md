# Introduction

## Using a Live Server with LiveWeave 

Before diving into the basics of HTML, setting up a live environment where you can see your changes in real time can enhance your learning experience. LiveWeave is an online tool that allows you to write HTML, CSS, and JavaScript and see the outcomes immediately. 

### Getting Started with LiveWeave 
 
- **Access LiveWeave:**  Open your browser and visit [LiveWeave](https://liveweave.com/) .
 
- **Write and Test Code:**  Use the panels for HTML, CSS, and JavaScript. Changes you make will be reflected in real time in the output panel.
 
- **Experiment and Learn:**  This tool provides a practical way to understand how HTML works interactively.



# Setting Up VS Code for Web Development 

Visual Studio Code (VS Code) is a powerful, lightweight code editor developed by Microsoft. It's popular among developers for its performance, customizability, and extensive extension ecosystem.

## Installing Visual Studio Code 

First, you need to download and install VS Code:
 
1. Visit the [VS Code website](https://code.visualstudio.com/) .

2. Download the version compatible with your operating system (Windows, macOS, or Linux).

3. Follow the installation prompts to install VS Code on your machine.

## Basic Configuration 
Once installed, open VS Code. You can customize the look and feel by navigating to `File > Preferences > Color Theme` or `File > Preferences > File Icon Theme`.
## Installing Essential Extensions 

Extensions in VS Code enhance its functionality. Here's how to install the recommended extensions for web development:

### Live Server 

Live Server launches a local development server with a live reload feature for static and dynamic pages.

1. Open VS Code.
 
2. Go to the Extensions view by clicking on the square icon on the sidebar or pressing `Ctrl+Shift+X`.

3. Search for "Live Server".
 
4. Click **Install**  on the Live Server extension by Ritwick Dey.

### JavaScript (ES6) Code Snippets 

This extension provides ES6 syntax code snippets for JavaScript development, making coding more efficient.

1. In the Extensions view, search for "JavaScript (ES6) code snippets".

2. Look for the extension by charalampos karypidis.
 
3. Click **Install** .

### Material Icon Theme 

The Material Icon Theme provides lots of icons based on the Material Design guidelines, making it easier to visually distinguish files and folders.

1. Search for "Material Icon Theme" in the Extensions view.
 
2. Click **Install**  on the Material Icon Theme by Philipp Kief.

### HTML CSS Support 

This extension ensures that class and ID names are autocompleted within your HTML and linked CSS files.

1. In the Extensions view, search for "HTML CSS Support".
 
2. Click **Install**  on the HTML CSS Support extension by ecmel.

## Using Live Server 

To use Live Server after installing:
 
1. Open your project folder in VS Code (`File > Open Folder`).

2. Right-click on an HTML file in the Explorer sidebar and select "Open with Live Server" or click "Go Live" from the status bar to launch the server.

3. Your default browser will open, and you can see your web page. Changes in the code will automatically refresh the page.
